# sealantern-plugins

FPS_Z 的 Sea Lantern 插件仓库。

## 插件列表

| 插件 | 分类 | 简介 |
|------|------|------|
| [event-director-ui](./event-director-ui) | 工具 | 事件导演台 — 带时间轴的剧情脚本编排工具 |

## 目录结构

```
sealantern-plugins/
└── event-director-ui/   # 事件导演台
    ├── manifest.json    # 插件声明
    ├── main.lua         # 插件逻辑
    └── style.css        # 界面样式
```

## 安装

在 Sea Lantern 插件市场搜索对应插件名安装，或从 [Releases](../../releases) 下载 zip 手动安装。
